## Feature map

### Implemented

- Device information
- Battery information
- `getprop` values
- USB mode detection
- Backup

### Partially Implemented (In progress)

- App management (almost done!)

### Coming Soon

- File Transfer
- Terminal

### Planned

- ROM Flashing
- Advanced boot
- Recovery update
